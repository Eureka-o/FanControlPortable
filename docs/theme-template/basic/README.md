# Basic 基础主题模板

这个版本适合做 THRM/FanControl 都尽量可用的主题。它只使用完整默认浅色变量、`.glacier-*` 组件钩子和通用 HTML 角色选择器，不依赖 FanControl 高级 `data-theme-card` / `data-theme-ui` 钩子。

## 适合做什么

- 改主色、背景色、文字色、卡片色。
- 改图表曲线颜色、温度/功耗颜色。
- 改左侧菜单、顶部栏、首页卡片、图表卡片。
- 演示左侧徽标：默认是文字徽标，也给了图片/SVG 徽标写法。

## 不适合做什么

- 不适合做复杂贴纸幕布、每张小卡片独立装饰、页面级强定制。
- 不建议放很多动画或大面积半透明模糊。
- 如果想做小八 Plus 那种完整视觉主题，请用 `advanced` 模板。

## 改名步骤

1. 把文件夹 `basic` 改成你的主题 ID，例如 `soft-blue`。
2. 打开 `theme.json`，把 `id` 改成同一个 ID。
3. 打开 `theme.css`，把所有 `html[data-theme="basic"]` 改成同一个 ID。
4. 如果你用 CSS 变量前缀，也把 `--basic-*` 改成自己的前缀，方便维护。

## 资源建议

- 小图标、徽标、简单装饰：优先 SVG，文件小、缩放清晰、颜色容易调。
- 插画、照片、纹理、背景幕布：优先 WebP，文件小、适合复杂图像。
- 不想画背景图：把 `--basic-background-image` 保持为 `none`，只用渐变即可。
