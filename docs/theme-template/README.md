# FanControl 主题模板

这里提供两个模板：

- `basic/`：基础版。只用默认浅色变量和 THRM/FanControl 常见 `.glacier-*` 组件，适合简单改色和尽量兼容 THRM。
- `advanced/`：高级版。基于小八 Plus 技术路线，包含本地资源、字体、贴纸、幕布、页面级背景、卡片透明和 FanControl 高级钩子。

完整制作规范见：

```text
docs/theme-diy-guide.md
```

推荐用法：

1. 新手或只想改颜色：从 `basic/` 复制一份开始。
2. 想做完整角色主题：从 `advanced/` 复制一份开始。
3. 分享主题时，把整个主题文件夹压缩发给别人，不要只发 `theme.css`。
