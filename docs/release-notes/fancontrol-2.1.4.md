# FanControl 2.1.4 发布说明

FanControl 2.1.4 是一个 WiFi 查找体验修复版本，主要改进设备 IP 查找、深度扫描提示和扫描进度显示。建议使用 WiFi 设备、电脑热点或动态 IP 兼容功能的用户更新。

## 主要修复

- 修复 WiFi 查找设备结束后用时可能显示为 `0ms` 的问题。
- 深度扫描现在会先检查常用局域网网段；如果仍未找到设备，会提示正在扩大搜索范围，并继续检查更多常见网段。
- 深度扫描过程中增加运行提示和预计进度，避免长时间扫描时界面看起来没有反应。
- 深度扫描过程中可暂停、继续或中止，适合临时切回游戏或发现网络设置不对时停止扫描。
- 优化设置页「查找设备 IP」区域显示，说明文字完整换行，深度扫描按钮会在查找按钮左侧自然出现。
- 扩大深度扫描覆盖范围，适合设备 IP 不在常见 `192.168.0.x`、`192.168.1.x` 或 Windows 热点网段时使用。
- 修复 WiFi 扫描相关中文和日语文案中可能显示为问号的问题。

## 使用建议

- 普通「查找 IP」仍然是优先选择，速度更快，适合设备和电脑处于同一常见网段的情况。
- 只有普通查找没有结果时再使用「深度扫描」。深度扫描会检查更多地址，耗时可能更长。
- 如果已经在路由器中给散热器设置了静态 IP，一般不需要开启「动态 IP 兼容」。
- 升级后会继续继承原有配置、曲线方案、保存的 WiFi IP、主题和设备档案。
