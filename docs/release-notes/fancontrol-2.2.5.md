# FanControl 2.2.5

FanControl 2.2.5 新增多套内置主题，并修复部分界面显示问题。建议 2.2.x 用户更新。

## 新增主题

- 新增 4 套内置主题：小八、Doro、耄耋、Cyberpunk 2077。
- 小八、Doro、耄耋为浅色主题，Cyberpunk 2077 为深色主题。
- 新主题会随安装包和便携版一起提供，安装或解压后可直接在设置页选择。

## 显示修复

- 修复 THRM 主题和部分自制主题左上角圆角区域可能出现黑块或露底的问题。
- 修复「温度与风扇历史详情」中 RPM 设备的风扇平均值单位显示错误。

## 升级说明

- 升级会保留原有设备、曲线、学习数据、WiFi IP、主题、托盘设置和开机启动设置。
- 如果之前遇到主题黑角或历史均值单位异常，升级后建议重启软件再查看。
