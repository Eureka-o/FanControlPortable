# FanControl 2.4.4

本次更新完善了应用内升级、主题管理和设备连接体验，同时进一步降低后台开销，让日常使用更加顺畅稳定。

## 应用更新

- 新增应用内下载与安装更新，无需手动查找安装包。
- 下载过程会实时显示进度，安装阶段也有清晰的状态提示。
- 安装完成后会自动重启 FanControl；更新失败时可直接重试或前往发布页下载。

## 主题与界面

- 设置页新增 Acrylic、Mica、Mica Alt 和关闭四种毛玻璃效果，可按系统与个人偏好选择。
- 统一毛玻璃外观下标题栏、侧栏和内容区的材质，减少区域之间的亮色接缝。
- 官方主题会随安装包放入安装目录的 `themes` 文件夹，便于查看和管理。
- 升级时不会覆盖已有主题，包括用户修改过的官方主题和自制主题。
- 优化左侧导航的选中质感，并为主要页面加入更自然的内容淡入效果。
- 修复传感器列表短暂刷新时无法稳定保留手动选择的问题。
- 首页风扇曲线预览会跟随当前主题的主色显示。
- 记住窗口位置、大小和最大化状态，下次启动时自动恢复。
- 减少不必要的界面刷新与重复工作，后台运行更轻量。

## 设备与智能控制

- 改善 BLE 与 HID 设备的连接、重连和退出稳定性。
- 改善休眠唤醒后的设备恢复流程，并修复部分中文传感器名称和错误信息显示异常的问题。
- 提升设备状态与设置读取的准确性，减少并发读取造成的异常结果。
- 温升预测默认启用；从旧版本升级时仍会保留用户原有选择。
- 智能控制只使用有效的实时设备数据进行学习，设备断开或休眠时不会写入无效样本。
- 改善系统休眠恢复和托盘状态自愈，长时间运行更稳定。

## 下载

- `FanControl-2.4.4-amd64-installer.exe`：安装版，支持后续在应用内直接更新。
- `FanControl-2.4.4-portable.zip`：便携版，解压后运行 `FanControl.exe`。
