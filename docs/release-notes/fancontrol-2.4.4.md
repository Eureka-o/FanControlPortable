# FanControl 2.4.4

本次更新改善毛玻璃外观，并提升设备连接与休眠恢复的稳定性。

## 界面体验

- 统一毛玻璃外观下标题栏、侧栏和内容区的材质，去除区域间的亮色接缝。
- 修复传感器列表短暂刷新时无法稳定保留手动选择的问题。
- 首页风扇曲线预览恢复为当前主题的主色显示。
- 记住窗口位置、大小和最大化状态，下次启动自动恢复。

## 稳定性

- 改进 HID 设备连接和休眠唤醒后的恢复流程。
- 修复部分中文传感器名称和错误信息可能出现乱码的问题。

## 下载

- `FanControl-2.4.4-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.4.4-portable.zip`：便携版，解压后运行 `FanControl.exe`。
