# FanControl 2.1.7 发布说明

FanControl 2.1.7 主要修复了温度桥接的偶发通信超时，并为 Windows 11 补上更自然的毛玻璃窗口效果。

## 主要更新

- 改进 TempBridge 通信方式，减少“桥接程序通信失败 / i/o timeout”这类偶发问题。
- 温度桥接读取失败后会更快重建，不容易卡在不可用状态。
- 桥接初始化日志更清晰，排查温度读取、PawnIO 和传感器枚举问题更方便。
- 错误提示和温度桥接异常提示新增“导出诊断日志”，反馈问题时可以直接保存 FanControl 专属诊断包。
- Windows 11 窗口加入 Mica 毛玻璃效果，整体外观更接近系统原生风格。
- 同步系统深浅色主题，避免材质和界面颜色不一致。
- 保留 2.1.6 的功耗显示、历史记录、设备配置和已有曲线方案。

## 升级建议

- 建议所有 2.0 / 2.1 用户更新到 2.1.7。
- 升级后会继续保留原有配置、曲线方案、保存的 WiFi IP、主题和设备档案。
