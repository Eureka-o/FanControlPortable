# FanControl 2.2.1

FanControl 2.2.1 是一个小修复版本，重点优化设备显示、托盘提示和部分界面细节。建议 2.2.0 用户更新。

## 连接与设备显示

- 启动后会优先沿用上次启用的设备档案，减少每次打开软件都回到默认设备的情况。
- 未连接设备时，首页也会尽量显示当前绑定的设备名称，而不是固定显示默认设备。
- 首页顶部设备信息、设置页、曲线页和后端当前设备现在使用更一致的选择逻辑，减少多设备切换后的显示不同步。
- 本版本不改变 WiFi 发包格式，也不会删除已有设备档案。

## 托盘显示

- 托盘右键菜单中的设备状态会跟随当前设备名称显示。
- 托盘悬停提示精简了文字顺序，把风扇速度提前显示，减少 Windows 截断提示时把 `RPM` 显示成 `R` 的情况。
- CPU/GPU 温度、功耗和风扇速度仍保留在托盘提示与右键菜单中。

## 界面细节

- 优化关于页面和部分右侧菜单的显示效果。
- 修复邮箱、QQ群等信息复制后的提示文案显示异常问题。

## 升级说明

- 升级后会继续保留原有配置、曲线方案、学习结果、保存的 WiFi IP、主题、托盘设置、开机启动设置和设备档案。
- 如果你正在使用多设备档案、RPM 设备，或经常通过托盘查看风扇状态，建议更新到 2.2.1。
