# FanControl 2.1.5 发布说明

FanControl 2.1.5 主要优化了设置页和设备连接页的显示体验，并为 WiFi 兼容设备加入了「智能启停」Beta 功能。

## 主要更新

- 新增 WiFi 设备「智能启停」Beta：可在退出、睡眠或休眠前将设备切换到待机速度，降低长时间保持原转速的情况。
- 智能启停设置改为折叠菜单，待机转速作为独立子项显示，界面更清晰。
- 优化「查找 IP」区域布局，减少按钮区空白，让普通扫描和深度扫描的入口更自然。
- 深度扫描支持暂停和中止，暂停时用时会停止累加，扫描图标也会停止旋转。
- 修复扫描结束后用时显示异常的问题。

## 使用建议

- WiFi「智能启停」仍为 Beta 功能，建议先保持默认的 1% 待机速度，并观察自己的设备是否能稳定响应。
- 如果普通「查找 IP」没有找到设备，可以按提示使用「深度扫描」。
- 升级后会继续继承原有配置、曲线方案、保存的 WiFi IP、主题和设备档案。
