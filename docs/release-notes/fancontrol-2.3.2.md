# FanControl 2.3.2

相比 2.3.1，FanControl 2.3.2 主要带来后台稳定性优化、主题显示修复，以及新的内置深色主题「沙丘」。建议 2.3.1 用户更新。

## 后台与稳定性

- 优化温度桥接组件的 GPU 状态检测流程，降低后台运行时对游戏帧率造成偶发影响的概率。
- 修复全局快捷键输入框在中文占位文字下字体不一致的问题。

## 主题与界面

- 新增内置深色主题「沙丘」，安装版和便携版会随包携带。
- 修复沙丘主题下设置页亮度、曲线学习目标温度等滑条的视觉一致性。
- 修复高级自定义主题在曲线页部分卡片圆角、背景和控件边界不一致的问题。
- 补全小八 Plus、蜡笔小新等内置主题字体中缺失的主题名和设备名字符。

## 性能提示

沙丘、小八 Plus、蜡笔小新等主题包含更多图片、半透明层和装饰元素，视觉效果更丰富，也可能带来更高的 GPU/渲染占用。如果使用核显、低功耗模式，或发现界面动画不够流畅，可以切换到默认浅色/深色主题。

## 下载

- `FanControl-2.3.2-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.3.2-portable.zip`：免安装便携版，解压后运行 `FanControl.exe`。
