# FanControl 2.4.1

相比 2.4.0，本版重点优化设备连接、后台运行表现和主题更新体验，并修复部分高级主题字体显示问题。

## 更新内容

- 优化设备连接体验：设置页的设备扫描与首页连接逻辑更加一致，蓝牙、HID 与 WiFi 兼容设备的发现和连接更清晰。
- 优化 WiFi 兼容模式：保留手动添加地址与动态 IP 兼容能力，扫描设备时会按当前兼容模式自动处理。
- 优化后台运行表现：减少后台空闲时不必要的刷新与重复状态更新，降低长时间挂后台时的资源占用波动，长时间运行时会更顺滑。
- 优化主题资源加载：高级主题的图片与字体资源改为按需加载，减少大主题带来的内存和启动压力。
- 提升连接稳定性：减少设备短暂无响应时反复重连造成的干扰，改善部分蓝牙设备健康检查后的状态保持。
- 优化主题升级体验：安装新版时不再简单删除并重装已有主题目录，用户自己添加的主题资源会更稳妥地保留。
- 修复高级主题部分中文缺字问题，补齐小八 Plus、蜡笔小新等高级主题在新增说明文字中的字体显示。
- 更新内置主题与主题模板的兼容信息，方便后续主题继续区分基础版和高级版能力。

## 下载

- `FanControl-2.4.1-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.4.1-portable.zip`：便携版，解压后运行 `FanControl.exe`。
