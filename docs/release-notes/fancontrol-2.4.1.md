# FanControl 2.4.1

相比 2.4.0，本版重点优化设备连接、后台运行流畅度、连接稳定性和主题更新体验。实际使用中，软件长时间挂在后台时会更安静，切回前台、加载主题和扫描设备时也会更顺滑。

## 更新内容

### 设备连接

- 优化设置页的设备连接区，扫描设备、当前设备、候选设备和兼容模式的展示更加清晰。
- 首页连接按钮与设置页扫描逻辑保持一致，避免两个入口行为不一致。
- 优化蓝牙、HID 与 WiFi 兼容设备的发现和连接流程，设备扫描结果会更稳定地跟随真实连接状态。
- 优化 WiFi 兼容模式，保留手动添加地址、动态 IP 兼容和深度扫描能力，并会按当前兼容模式自动处理扫描范围。

### 流畅度与后台占用

- 优化后台空闲状态下的刷新策略，减少不必要的状态更新和重复轮询，降低资源占用波动。
- 优化温度历史、配置读取和设备状态刷新时的重复工作，长时间运行时更不容易出现卡顿感。
- 优化高级主题资源加载方式，图片和字体等资源改为按需加载，减少大主题带来的启动压力和内存压力。
- 热修复：减少程序内嵌的高级主题资源，降低常驻内存占用；高级主题仍会随安装包和便携包的 `themes/` 目录提供。
- 优化前端心跳与状态更新节奏，减少隐藏到后台时的无效刷新。

### 连接稳定性

- 改善设备短暂无响应时的处理方式，减少单次抖动导致的误断连和反复重连。
- 优化部分蓝牙设备健康检查后的状态保持，避免刷新状态时覆盖已有控制状态。
- 优化 HID 设备打开失败时的错误信息，便于后续定位用户侧设备连接问题。

### 主题与升级体验

- 优化主题升级策略，安装新版时不再简单删除并重装已有主题目录，用户自己添加的主题资源会更稳妥地保留。
- 修复高级主题部分中文缺字问题，补齐小八 Plus、蜡笔小新等高级主题在新增说明文字中的字体显示。
- 更新内置主题与主题模板的兼容信息，方便后续主题继续区分基础版和高级版能力。

## 下载

- `FanControl-2.4.1-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.4.1-portable.zip`：便携版，解压后运行 `FanControl.exe`。
