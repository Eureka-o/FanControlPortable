# FanControl 2.5.1

本版本重点优化软件内更新、运行稳定性和曲线编辑体验。

## 软件内更新

- 下载进度改为可拖动的右上角浮窗，可收起为带百分比的进度圆环，切换页面不会中断下载。
- 新增暂停、继续、中止和失败重试；中止后会清理未完成的下载文件。
- 支持断点续传，并在代理连接不可用时自动尝试直连，降低网络波动导致的重复下载。
- 更新临时文件统一存放在 FanControl 安装目录的 `updates` 文件夹中，安装完成后自动清理。
- 安装完成后自动重新启动 FanControl，并在新版首次启动时显示更新完成提醒。

## 稳定性与响应

- 优化休眠和退出前的设备写入收尾，正在等待的 WiFi、串口或 BLE 操作可以及时取消，减少系统休眠被网络超时拖慢的情况。
- 优化配置并发读写，避免智能控制学习结果覆盖刚刚修改的主题、传感器或设备设置。
- 深度设备与 WiFi 扫描使用独立通信通道，扫描期间的常规状态刷新和设置操作保持可响应。
- 配置读取失败会明确提示错误，不再用默认配置掩盖核心服务或数据异常。

## 曲线与界面

- 离开存在未保存修改的曲线页时会继续使用保存、放弃或取消确认，避免误丢曲线草稿。
- 设备切换期间只接收最新一次状态刷新结果，减少旧设备数据覆盖当前页面的情况。
- 启动时不再加载被截断的自定义主题缓存，避免短暂出现残缺样式。
