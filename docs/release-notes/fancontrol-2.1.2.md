# FanControl 2.1.2 Release Notes

FanControl 2.1.2 是 2.1.x 的安装器热修复版本，建议所有 2.1.0 / 2.1.1 用户更新。

## 下载

- `FanControl-2.1.2-amd64-installer.exe`：推荐大多数 Windows 用户使用。
- `FanControl-2.1.2-portable.zip`：免安装便携版，解压后运行 `FanControl.exe`。

## 修复内容

- 修复部分用户从 2.1.0 / 2.1.1 升级时，安装器因系统中已经存在 PawnIO 而弹出 PawnIO 安装失败提示的问题。
- 安装器现在会检测已安装的 PawnIO，并在普通软件升级时跳过驱动覆盖安装。
- 如果确实需要修复 PawnIO，请在 FanControl 设置页中使用“重新安装 PawnIO”，不要依赖普通升级流程覆盖共享驱动。

## 升级说明

- 从 2.0 / 2.1 升级到 2.1.2 会继续保留已有曲线、WiFi IP、主题、学习数据和设备配置。
- 本版本不改变设备控制逻辑，不包含 2.2.0 的设备导入和设置页重构内容。
- 如果你在 2.1.0 / 2.1.1 升级时遇到 PawnIO 已安装导致的弹窗或安装中断，请直接安装 2.1.2。
