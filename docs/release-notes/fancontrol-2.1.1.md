# FanControl 2.1.1 发布说明

这是一个建议所有用户更新的修复版本。
本次更新主要改善学习曲线、百分比/RPM 设备切换、手动速度控制和设备功能显示，让 2.1.x 在日常使用中更稳定、更不容易显示错单位或误用旧设备配置。

## 下载

- `FanControl-2.1.1-amd64-installer.exe`：推荐大多数 Windows 用户使用，适合正常安装和从旧版本升级。
- `FanControl-2.1.1-portable.zip`：免安装便携版，解压后运行 `FanControl.exe`。

## 建议更新的用户

- 正在使用 FanControl 2.0 或 2.1 的用户。
- 使用过学习曲线，或发现学习曲线百分比显示不符合预期的用户。
- 在百分比设备和 RPM 设备之间切换过的用户。
- 导入过设备档案，或正在尝试 WiFi、BLE、虚拟串口/COM、HID/RPM 设备档案的用户。

## 本次修复

- 学习曲线的百分比显示更准确。百分比学习数据会按真实百分比显示，不再把内部刻度直接当成百分比展示。
- 切换百分比设备和 RPM 设备后，首页、曲线、手动档位、自定义速度、标题栏状态等位置会跟随当前设备显示正确单位和范围。
- RPM 设备的手动/自定义速度控制恢复正常。切换设备后，软件会按当前启用设备发送速度，不再被上一个百分比设备的配置影响。
- 切换设备或连接方式后，旧设备上报的数据不会再覆盖当前设备的速度单位。
- RPM 设备不会再错误沿用旧的 0-100 百分比曲线作为 0-4000 RPM 曲线。
- 不支持灯光、通电自启、智能启停、原始命令等功能的设备，不会再因为连接方式相同就显示这些控制项。
- 默认 WiFi 设备不再对不支持的灯光、智能启停、亮度、RGB 等功能返回“成功”，避免造成误解。

## 升级说明

- 从 FanControl 2.0/2.1 升级会保留已有配置，包括 WiFi IP、曲线方案、学习数据、用户设备档案和导入主题。
- 使用安装版的用户可以直接运行新版安装包升级。
- 使用便携版的用户，请先从托盘退出 FanControl，再解压新版覆盖旧文件夹。不要删除原来的 `config` 目录。
- 用户可见软件名仍为 `FanControl`，仓库和更新地址仍为 `Eureka-o/FanControlPortable`。

## 后续计划

WiFi 智能启停和 BLE 智能启停会放到后续版本继续完善。因为这类功能会直接影响设备启停和散热安全，我们会在加入能力判断、设备探测和失败回退后再开放给用户使用。
