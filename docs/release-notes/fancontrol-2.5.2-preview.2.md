# FanControl 2.5.2 Preview 2

这是 2.5.2 的第二版功能预览，继续验证软件内更新的下载安全性与断点续传稳定性，并改进下载进度浮窗。

## 下载进度浮窗

- 修复进度浮窗收起为圆环后，部分情况下点击圆环无法重新展开的问题。
- 加宽展开后的下载进度卡片，使进度信息和暂停、中止等操作更舒展。
- 保持收起态和展开态均可拖动，切换页面不会中断当前下载任务。

## 软件内更新

- 修复同一基础版本下预览版本号的比较逻辑，Preview 1 现在可以正确识别 Preview 2 为新版本。
- 使用 GitHub Release 资产提供的 SHA-256 摘要校验安装包，校验失败时不会启动安装，并会清理无效断点文件。
- 下载前会识别目标版本：属于当前目标且服务器校验信息一致的断点文件可以继续下载，其他版本的断点缓存会自动清理并重新开始。
- 暂停下载不再占用下载总超时时间，长时间暂停后仍可继续当前任务。
- 保留代理优先、失败后尝试直连的下载策略，并为连接响应设置独立超时。

## 缓存清理

- 启动时清理旧版遗留在系统临时目录中的更新文件。
- 自动清理安装目录 `updates` 中超过 7 天的已知更新残留，同时保留近期断点文件和无关文件。

此版本为预览版，建议重点验证进度圆环的收起与展开、拖动、暂停后继续、下载中断后重试以及安装完成后的自动重启流程。
