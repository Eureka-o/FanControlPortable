# FanControl 2.3.0

FanControl 2.3.0 重点更新主题系统与界面质感，并修复开机自启动状态同步问题。建议 2.2.x 用户更新。

## 新增主题

- 新增 2 套内置主题：小八 Plus、蜡笔小新。
- 新主题会随安装包和便携版一起提供，安装或解压后可直接在设置页选择。
- 主题资源默认放在软件目录的 `themes` 文件夹，便携版也会随包携带完整主题文件。

## 界面优化

- 优化首页、曲线页和设置页的卡片层级、圆角、阴影和透明质感，让默认主题与自制主题的视觉更统一。
- 优化首页风扇曲线缩略图，显示范围与当前 20°C 起始曲线逻辑保持一致。
- 优化开关、下拉选择、复选项和部分悬停状态，鼠标经过时更容易判断当前可点击区域。
- 调整曲线页部分区域的样式，减少不必要的小卡片感，让页面结构更接近实际功能分区。

## 主题兼容

- 自定义主题支持使用主题文件夹内的相对图片、字体和装饰资源，不再需要把大图片全部写进 CSS。
- 安装版和便携版都会优先使用软件目录下的主题文件，旧版本留在用户目录的主题会继续尽量迁移。
- 继续保留 THRM 风格的主题结构：`theme.json`、`theme.css` 和同目录资源文件。

## 修复

- 修复开机自启动设置页显示状态可能与 Windows 实际自启动状态不同步的问题。
- 修复部分主题下首页曲线缩略图、卡片圆角和控件 hover 状态不一致的问题。
- 修复自定义主题资源随安装包/便携版携带时可能缺图或引用失效的问题。

## 性能提示

- 小八 Plus、蜡笔小新包含更多图片、半透明层和装饰元素，视觉效果会比默认主题更丰富，也可能带来更高的 GPU/渲染占用。
- 如果使用核显、低功耗模式，或发现界面动画不够流畅，可以切换到默认浅色/深色主题，或使用较轻量的自定义主题。

## 升级说明

- 升级会保留原有设备、曲线、学习数据、WiFi IP、主题、托盘设置和开机启动设置。
- 便携版更新时，请先从托盘退出 FanControl，再解压新版覆盖旧文件夹；不要删除原来的 `config` 目录。
