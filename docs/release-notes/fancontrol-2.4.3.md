# FanControl 2.4.3

本次更新聚焦设备连接显示和趋势图可读性，让启动后的状态更准确、日常查看更直观。

## 设备连接

- 修复 BS1 已连接时，首页偶尔显示为 WiFi 设备的问题。
- 改善启动和自动重连后的设备状态保持，设备名称与实际连接保持一致。

## 趋势图与界面

- 优化趋势图坐标刻度，风扇转速、温度和功耗更容易区分。
- 首页风扇曲线预览会跟随当前主题的风扇颜色显示。

## 下载

- `FanControl-2.4.3-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.4.3-portable.zip`：便携版，解压后运行 `FanControl.exe`。
