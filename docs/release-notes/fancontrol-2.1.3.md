# FanControl 2.1.3 发布说明

FanControl 2.1.3 主要改进 WiFi 设备查找和动态 IP 兼容，适合使用电脑热点、局域网地址可能变化，或需要重新找回散热器 IP 的用户。

## 主要更新

- 新增 WiFi「查找 IP」。在已有 WiFi 设备档案的基础上，软件会优先检查当前保存地址、同网段地址、当前电脑网卡网段和 Windows 热点网段。
- 新增「深度扫描」。普通查找没有结果后才显示，会继续检查常见路由器和设备热点网段。
- 新增 WiFi「动态 IP 兼容」。开启后，当连接失败时，软件会只改变当前保存 IP 的最后一位进行查找；找到设备后自动更新该设备的 IP 属性。
- 设备未连接时也可以查找 IP；但软件不会在没有已保存 WiFi 设备档案的情况下盲扫未知设备。
- 设备身份仍然以已保存设备名称为准，例如 `Slim压风散热器Pro`；IP 只是这个设备的连接属性，可以被手动修改或扫描更新。
- BLE/HID 连接方式仍保留给已导入或手动配置的设备；未完成验证的飞智设备信息继续隐藏，做好后再开放。

## 使用建议

- 如果已经在路由器中给散热器设置了静态 IP，一般不需要开启「动态 IP 兼容」。
- 本版本不加入 WiFi 智能启停；该功能需要固件接口或更完整的软件安全策略，会放到后续版本。
- 建议所有 2.0/2.1 用户更新。升级后应继续继承原有配置、曲线方案、保存的 WiFi IP、主题和设备档案。
