# FanControl 2.4.0

相比 2.3.2，FanControl 2.4.0 主要重做了设备扫描与连接入口，新增默认浅色/深色外观的玻璃材质开关，并合并了原计划 2.3.3 中的主题显示、托盘恢复和后台稳定性修复。

## 设备连接

- 设置页新增统一的「设备连接」区域，扫描、发现、连接和当前设备状态集中展示。
- 首页连接按钮与设置页扫描逻辑保持一致，避免 WiFi 兼容设备被已连接的蓝牙或 HID 设备抢占。
- 「扫描设备」默认扫描内置 HID/BLE 设备；开启兼容模式后，会同时扫描 WiFi 和虚拟串口/COM 兼容设备。
- WiFi 兼容模式支持浅扫描、深度扫描、动态 IP 兼容和手动添加地址；扫描结果会逐步显示。
- 未连接设备时，首页和设置页统一显示未连接状态，不再误显示保存过的兼容设备名称。

## 界面与主题

- 默认浅色/深色外观新增可切换的 WindowGlass 玻璃质感，侧边栏、顶部栏和主要卡片会使用半透明毛玻璃层级；自定义主题不受该开关强制影响。
- 页面切换恢复参考软件的淡入淡出节奏，并增加轻量的卡片错峰上浮动画。
- 修复沙丘主题下设置页亮度、曲线学习目标温度等滑条的视觉一致性。
- 修复高级自定义主题在曲线页部分卡片圆角、背景和控件边界不一致的问题。
- 优化部分主题页面中说明文字过早换行、可读性不佳的问题。
- 补全小八 Plus、蜡笔小新等内置主题字体中缺失的主题名、设备名字符。

## 后台与稳定性

- 优化后台 GPU 状态探测流程，减少温度桥接组件在后台运行时偶发卡顿或占用升高的情况。
- 提升 Windows 开机自启动或 Explorer 重启后的托盘图标恢复稳定性。
- 修复全局快捷键输入框的中文占位文字字体不一致问题。

## 性能提示

沙丘、小八 Plus、蜡笔小新等主题包含更多图片、半透明层和装饰元素；默认玻璃外观也会使用更多透明与模糊效果。如果使用核显、低功耗模式，或发现界面动画不够流畅，可以关闭玻璃材质或切换到更轻量的主题。

## 下载

- `FanControl-2.4.0-amd64-installer.exe`：安装版，推荐大多数用户使用。
- `FanControl-2.4.0-portable.zip`：免安装便携版，解压后运行 `FanControl.exe`。
